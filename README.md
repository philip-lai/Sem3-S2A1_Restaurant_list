# 專案名稱 : 餐廳清單
# 功能簡介
* 檢視各間餐廳相關資訊
* 透過以下關鍵字搜尋餐廳
  * 餐廳中文名稱
  * 餐廳英文名稱
  * 餐廳地址
  * 餐廳種類
  * 餐廳敘述

# 安裝說明
#### 1. 開啟終端機，將專案下載至本機端

      git clone https://github.com/philip-lai/Sem3-S2A1_Restaurant_list.git
#### 2. 切換檔案路徑至專案資料夾，並安裝套件

      cd Sem3-S2A1_Restaurant_list
      npm install
#### 3. 執行專案
      npm run dev
      

# 使用工具
* visual studio
* Express
* Express-handlebars
* node-js
